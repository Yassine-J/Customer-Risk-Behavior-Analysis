# 📊 Customer Churn Analysis (Banking)

## 🎯 Objectif
Analyser les comportements clients pour identifier les profils à risque et améliorer la rétention.

## 🛠️ Outils
- Python (Pandas)
- Power BI

## 📈 Analyses
- Analyse du churn
- Segmentation clients
- Identification des facteurs de départ

## 💡 Insights
- Certains profils présentent un risque élevé de départ
- Le churn est lié à des facteurs comportementaux

## 🚀 Recommandations
- Mettre en place des stratégies de fidélisation
- Cibler les clients à risque
